# YeBetWeg Quick Start Guide

## What is YeBetWeg?

**YeBetWeg (የቤት-ወግ)** = "Way of Building" in Amharic

A complete construction knowledge platform for Ethiopia that connects:
- 🏠 Homeowners with builders
- 🔨 Professionals with clients  
- 💰 Sellers with buyers
- 📚 Knowledge seekers with experts

---

## First Time Users (30 Seconds)

1. **Pick your language**: Click AM/EN in top-right
2. **Explore sections**: Use navigation menu (Knowledge/Tips/Prices/Marketplace/Professionals)
3. **Browse free content**: All core features available without signup
4. **Contact us**: Use Contact section at bottom anytime

## I Want To...

### Read about Construction
👉 Click **Knowledge Hub** → Browse or filter by category (Construction/Philosophy/Market Insights/Regulations/Materials)

### Get Expert Tips
👉 Click **Tips** → See all 10 tips (6 free, 4 premium) → Unlock premium for full access

### Check Material Prices
👉 Click **Market Prices** → Select category → View current Addis Ababa prices with trends

### Buy/Sell/Rent
👉 Click **Marketplace** → Select category → View listings with verified badges → Click "Contact" to inquire

### Hire a Professional
👉 Click **Professionals** → Filter by specialty → Click "Hire" → Fill quick form → Professional contacts you

### List My Property
👉 Click **Marketplace** → Click "List Your Property or Service" → Fill form → Submit

### Join as Professional
👉 Click **Professionals** → Click "Join as Professional" → Fill registration → Wait for approval

### Upgrade to Premium
👉 Click **Premium** → Select plan (500 or 1200 ETB/month) → Pay via Chapa → Access premium features

## Common Issues

**Images not loading?**
- Check internet connection
- Try different browser
- Clear cache: Ctrl+Shift+Delete

**Can't switch language?**
- Click Globe icon (AM/EN) in navbar
- Try refreshing page
- Check if JavaScript enabled

**Marketplace inquiry not working?**
- Fill all required fields
- Try different browser
- Check if cookies enabled

**Payment failed?**
- Verify sufficient account balance
- Try alternate payment method
- Contact Chapa support
